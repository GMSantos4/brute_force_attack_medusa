# Ataque de Força Bruta utilizado MEDUSA

O objetivo deste repositório é armazenar o desafio de projeto do Bootcampo Santander 2025 - Cibersegurança.
Conforme sugerido no desafio, esta entrega será uma descrição detalhada do que foi aprendido nas aulas com exemplos de código e reflexões.
Para o leitor que queira ir direto ao ponto, ou seja, onde os códigos e configurações são efetivamente aplicadas, comece pelo tópico **Primeiros Contatos com Metasploitable2**.
